# Реализация кастомных структур данных (Car Collections)

Учебный проект по реализации базовых структур данных с нуля на языке Java. В рамках проекта разработаны собственные аналоги стандартных коллекций Java (аналоги `HashSet` и `LinkedList`) с полным покрытием модульными тестами (Unit-тестирование).

Проект демонстрирует глубокое понимание принципов ООП, структуры памяти JVM, алгоритмов работы хеш-таблиц (разрешение коллизий методом цепочек, рехеширование) и двусвязных списков, а также практическое применение принципов TDD (Test-Driven Development).

---

## Технологический стек

*   **Язык программирования:** Java (совместим с актуальными версиями)
*   **Тестовый фреймворк:** JUnit 5
*   **Сборщик проектов:** Maven
*   **Концепции и паттерны:** ООП, SOLID (ISP/LSP), двусвязные списки, хеширование, итераторы (шаблон *Iterator*).

---

## Архитектура проекта

Проект спроектирован с четким разделением на интерфейсы (контракты) и их конкретные реализации для обеспечения слабой связанности (Loose Coupling):

```text
src/
├── main/java/org/example/Collections/
│   │
│   ├── Car.java                # Доменная модель (с переопределенными equals & hashCode)
│   │
│   # --- Слой интерфейсов (Контракты) ---
│   ├── CarCollection.java      # Базовый контракт коллекции, расширяющий Iterable
│   ├── CarList.java            # Расширение контракта для работы со списками по индексам
│   ├── CarSet.java             # Расширение контракта для работы с уникальными множествами
│   │
│   # --- Слой реализаций (Структуры данных) ---
│   ├── CarLinkedList.java      # Кастомный двусвязный список (Double-linked list)
│   └── CarHashSet.java         # Кастомная хеш-таблица с методом цепочек (Chaining)
│
└── test/java/org/example/Collections/
    # --- Слой модульного тестирования ---
    ├── CarLinkedListTest.java  # Unit-тесты для двусвязного списка
    └── CarHashSetTest.java     # Unit-тесты для хеш-множества
